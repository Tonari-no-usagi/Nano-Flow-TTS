## プロジェクトについて
Nano-Flow-TTSは、Rustと言語学的な知見を組み合わせた、軽量・高品質な音声合成エンジン (TTS) の構築を目指す個人プロジェクトです。

### 開発の目的
- **学習・研究**: モダンな音声合成アルゴリズムである Flow Matching (Rectified Flow) や、Rustによるプロジェクト構成の学習を目的としています。
- **軽量化の追求**: 低スペック環境（VRAM 8GB以下など）でも動作する、効率的な階層型モデル構造の実装に挑戦しています。

### ライセンスと公開範囲
- **非OSS (Not Open Source Software)**: 本プロジェクトは個人の学習目的で作成されており、オープンソースソフトウェアとして公開・配布・再利用を許可しているものではありません。

---

## 主な特徴
- **アルゴリズム**: Optimal Transport Conditional Flow Matching (OT-CFM) を採用。
- **アーキテクチャ**:
  - **Encoder**: 畳み込みに基づいた軽量な音素エンコーダー。
  - **Hierarchical Decoder**: 16次元(Coarse)と64次元(Fine)に分けた階層的なメルスペクトログラム生成。
- **言語支援**: 日本語の形態素解析 (`jpreprocess`) に基づくアクセント情報付き音素トークナイザー。

## 構成
- `nflow-core`: 共通のデータ構造とトークナイザー
- `nflow-text`: テキスト解析・音素変換（日本語）
- `nflow-model`: Flow Matching モデル本体
- `nflow-train`: 学習用パイプライン
- `nflow-synth`: 音声合成エンジン・Griffin-Lim デコーダー
- `nflow-gui`: ユーザーインターフェース (開発予定/途中)
